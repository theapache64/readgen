## 🛠 Installation

```
sudo npm install -g readgen-cli
```

## ⌨️ Usage

```shell script
Usage: read-gen [OPTIONS]

Options:
  -f TEXT     Custom font
  -s INT      Font size
  -c          Cover only
  -h, --help  Show this message and exit
```

- To generate README.md, install.sh and cover image

```shell script
~$ readgen
```

- To generate cover image alone

```shell script
~$ readgen -c
```

## 🥼 Run tests

```shell script
./gradlew test
```
